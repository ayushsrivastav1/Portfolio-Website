# Portfolio Website

A modern, responsive **Portfolio Website** built using **React.js** and **Tailwind CSS** to showcase my projects, skills, and experience as a Frontend Developer.

## Live Demo
🔗 https://ayushsrivastav-portfolio.netlify.app/

## Tech Stack

- React.js
- Tailwind CSS
- HTML5
- JavaScript (ES6+)
- Vite
- Netlify (Deployment)

## Features

- Fully responsive design (Mobile, Tablet, Desktop)
- Clean & modern UI
- Smooth hover & transition effects
- Projects section with live demo & GitHub links
- Resume download option
- Contact section / social links
- Fast performance & optimized build
